MPEG-7 Classification Example using Deep Learning and Fine Tunning

In this page, we aim to classify the images from MPEG7 Shape database using neural network library Keras. The pre-trained model chosen for image classification is InceptionV3[1] with weights trained on ImageNet.


The following steps are necessary to utilize this model in our application:

  • Install SARpy's requirements: scikit-learn (http://scikit-learn.org/) and NumPy
  • Install `pandas` and/or `matplotlib` for better visualization of results (Optional)
  • Install Keras (https://keras.io/)
  • Download the weights files for the pre-trained network(s) (which well be done automatically when you import and instantiate the respective network architecture)
  • Train the model with smaller set on dataset images (Fine Tunning)
  • Apply the pre-trained ImageNet networks from previous step to the images on database.

The MPEG-7 Shape dataset is avaliable in: http://www.dabi.temple.edu/~shape/MPEG7/dataset.html, and also through SARpy's datasets module.

Data Loading, Normalization and Preprocessing¶

From our implemented library, we load the MPEG7 dataset images through the function datasets.load_mpeg7. The InceptionV3 model requires all images to be of the same dimensions and have three channels; in the original model, the required dimensions were 299x299x3. We resize all of the dataset images to these dimensions, and convert them to three-channel RGB.

Model Training¶

The InceptionV3 model has trained weights for image classification on ImageNet. To fit this model for the new task (identify our own dataset images), we replace the top, fully-connected layers of the original model, and train these new layers with our own data. This is called fine-tuning a pre-trained network.

Further information is available on https://keras.io/applications/#inceptionv3.

In [6]:
# loading the base pre-trained model, without the top layers
base_model = inception.InceptionV3(weights='imagenet', include_top=False)

# Now we replace the top layers with our own.
# First, we add a pooling layer from the output of the existing model:
x = base_model.output
x = GlobalAveragePooling2D()(x)

# Then, we add a 1024-neuron fully connected layer - this is the layer
# that will learn to predict our classes, hopefully
x = Dense(1024, activation='relu')(x)

# Finally, we add a 70-neuron layer, where each neuron corresponds to
# to a single class probability
predictions = Dense(number_of_classes, activation='softmax')(x)

# This is the model we will train: the same input as the base model,
# but outputting to our new top layers 
model = Model(inputs=base_model.input, outputs=predictions)

# We want to train only the top layers (which were randomly initialized)
# i.e. freeze all convolutional InceptionV3 layers
for layer in base_model.layers:
    layer.trainable = False

# Now, we compile the model with a RMSProp optimizer, and with cross-entropy
# as our loss function.
model.compile(optimizer='rmsprop', loss='categorical_crossentropy', metrics=['accuracy'])

# train the model on the new data for a few epochs
model.fit(X_train,y_train,batch_size=32, epochs=10);
