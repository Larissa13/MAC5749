sarpy.datasets package¶

Submodules¶

  • sarpy.datasets.datasets module

Module contents¶

This module contains utilities to load datasets, and includes some in-built datasets. Based on: https://github.com/scikit-learn/scikit-learn/tree/master/sklearn/datasets

sarpy.datasets.load_mpeg7()¶

Loads the MPEG7 Shape dataset, containing 1400 images of 70 classes (20 images per class). Returns ——- dataset : dict

Dictionary object, the interesting attributes are: ‘bitmaps’, a vector of the images of the dataset as 2D NumPy arrays, and ‘targets’, a vector of the target/class of each image.
