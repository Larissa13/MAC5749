sarpy.shape_scale(shape, c, center=(0, 0))¶

Scales shape by a certain factor. This function calls the appropriate shape_scale function for the type of the given shape.

Parameters: * shape: Shape

  • Input shape
  • c: {float, tuple of floats}
    • Scale factors. Separate factors can be defined as (row_scale, col_scale)
  • center: tuple of ints, optional
    • (x,y)-coordinates of the center of the image

Returns: * scaled_shape: Shape

  • Scaled version of the input shape
sarpy.shape_shift(shape, c)¶

Shifts shape by a certain factor. This function calls the appropriate shift_scale function for the type of the given shape.

Parameters: * shape: Shape

  • Input shape
  • c: {float, tuple of floats}
    • Shift factors. Separate factors can be defined as (row_scale, col_scale)

Returns: * shifted_shape: Shape

  • Shifted version of the input shape
sarpy.convert_shape(shape, new_type, **kwargs)¶

Converts a given shape into another representation.

Parameters: * shape: Shape

  • Input shape
  • new_type: int - New type of the shape. Must be one of the types defined in sarpy.utils.
  • kwargs - Arguments for the different conversions.

Returns: * converted_shape: Shape

  • Converted representation of the input shape
